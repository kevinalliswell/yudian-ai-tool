# 01 · 产品功能规格

## 支持设备

| 型号    | MODEL 十进制 |
| ------- | -----------: |
| AI-516  |         5160 |
| AI-516P |         5167 |
| AI-518  |         5180 |
| AI-518P |         5187 |

## 四个主界面

1. 连接管理：扫描串口、选择端口、从站地址 1-80、连接/断开、型号识别。
2. 实时监控：PV/SV/MV、当前小数点、PV 时序图、离线降级。
3. 参数设置：SP1、PID、运行/保持/停止。
4. 温控曲线：本地预设、段编辑、上传、下载。段数 1-50，温度 -200~1800 ℃，时间 >=0 分钟。

默认串口参数：9600 baud、8 数据位、2 停止位、无校验。

非目标：多仪表同时连接、历史报表导出、完整参数全集编辑。
